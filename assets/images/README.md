# Images

Place images in this directory and its sub-directories.

## Structure

```
assets/images/
├── event.jpg               # Main hero image used on the home page
├── gallery/
│   ├── 2026-01.jpg         # Past event photos (named YEAR-NN.jpg)
│   ├── 2026-02.jpg
│   └── ...
└── sponsors/
    ├── smiths-garden-centre.png
    ├── old-bell-inn.png
    └── ...                 # One PNG/JPG per sponsor (transparent background preferred)
```

## Notes

- Hero image: ideally landscape, at least 1200 × 500 px.
- Gallery photos: any ratio; displayed at 180 px height in the grid.
- Sponsor logos: ideally PNG with transparent background, max 160 × 80 px display size.
